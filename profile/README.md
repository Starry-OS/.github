这里是 [StarryOS](https://github.com/Azure-stars/Starry) 组件集中存放的组织。我们期望做到：

- 内核组件细粒度划分
- 每一个内核组件作为一个独立的仓库，有对应的集成 CI 测试
- 可以通过统一的工具在主仓库直接拉取对应所需的仓库，打包成为一个整体内核进行运行
- 每一个子组件在足够细粒度的情况下，支持复用到其他内核
